
Задание 1.	CRUD (create, read, update, delete).
У вас есть всего 1 табличка User.

Необходимо реализовать стандартное crud приложение, которое отображаем список всех юзеров в базе (с пейджингом). С возможностью их удаления, редактирования, добавления новых. И поиска по уже существующим.
По какому полю искать – каждый решает для себя сам. Можно

1. После деплоя приложение запускается по адресу: http://localhost:8080/JavaRushCRUD/index
2. Поиск, редактирование и удаление происходит по ID юзера.
3. После выполнения CRUD операций требуется нажать кнопку UpdateURL для обновления адреса приложенияи просмотра таблицы юзеров.
 ограничиться полем name, можно реализовать фильтр для любого поля. 
4. Проект сделан в IDE Eclipse.
